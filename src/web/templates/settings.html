How fallback works
LLM CLI fallback

If the configured primary CLI returns an error, times out, or is not installed, AutoApply will try the fallback CLI when auto-fallback is enabled.

Feature-level fallback

Even after both CLIs fail, parts of the system still degrade gracefully: JD parsing falls back to regex, cover letters fall back to a deterministic template, bullet rewrites keep the original bullet, and QA falls back to templates or manual review.

Authentication

Both CLIs authenticate independently. Install and sign in locally before relying on LLM-backed parsing or generation.
